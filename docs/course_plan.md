# Программа курса по языку программирования C++

Программа курса разработана для студентов 2-го курса факультета КФ МГУ имени М.В Ломоносова. Курс предполагает знание основ программирования на языке C, структур данных и алгоритмов.

- ### Особенности языка C++ и области его применения

- ### Компиляторы C++ и IDE
